# 拾艺局·开发规划

## 项目概述

| 项目 | 说明 |
|------|------|
| **项目名称** | 拾艺局 (ShiYiJu) |
| **项目类型** | 高端艺术品社交电商平台 |
| **目标用户** | 艺术家、经纪人、收藏家、艺术推广者 |
| **交付形态** | 微信小程序 + iOS APP + Android APP |

---

## 技术架构

### 前端技术栈

| 技术 | 说明 | 版本 |
|------|------|------|
| **Uni-app** | 跨平台框架 | 4.x |
| **Vue 3** | 前端框架 | 3.4+ |
| **uView UI** | UI 组件库 | 2.x |
| **Pinia** | 状态管理 | 2.x |
| **uni-ui** | uni-app 官方组件库 | 最新 |

### 后端技术栈

| 技术 | 说明 | 版本 |
|------|------|------|
| **Spring Cloud Alibaba** | 微服务框架 | 2023.x |
| **Spring Boot** | 基础框架 | 3.2.x |
| **MySQL 8.0** | 关系型数据库 | 8.0 |
| **Redis 7.x** | 缓存/分布式锁 | 7.x |
| **MongoDB 6.x** | 社区内容存储 | 6.x |
| **RocketMQ** | 消息队列 | 5.x |
| **Spring Cloud Gateway** | API 网关 | 最新 |

### 基础设施

| 组件 | 说明 |
|------|------|
| **对象存储** | 腾讯云 COS / 阿里云 OSS |
| **CDN** | 腾讯云 CDN / 阿里云 CDN |
| **短信服务** | 腾讯云 SMS / 阿里云 SMS |
| **容器化** | Docker + Kubernetes |

## 项目结构

```
shiyiju/
├── backend/                      # 后端项目
│   ├── shiyiju-gateway/         # API 网关服务
│   ├── shiyiju-common/          # 公共模块（实体、工具、配置）
│   ├── shiyiju-user/            # 用户服务（认证、用户信息）
│   ├── shiyiju-product/         # 商品服务（作品管理）
│   ├── shiyiju-order/           # 订单服务（交易流程）
│   ├── shiyiju-auction/          # 拍卖服务
│   ├── shiyiju-promotion/        # 分销服务（艺荐官、佣金）
│   ├── shiyiju-community/        # 社区服务（艺术圈）
│   ├── shiyiju-message/          # 消息服务
│   └── shiyiju-file/            # 文件服务（上传、存储）
│
├── frontend/                     # 前端项目
│   ├── shiyiju-app/             # Uni-app 主项目
│   └── shiyiju-admin/           # 运营后台（可选）
│
├── documents/                    # 文档
│   ├── api/                     # API 文档
│   ├── db/                      # 数据库设计
│   └── guide/                   # 开发指南
│
└── deploy/                       # 部署配置
    ├── docker/
    └── k8s/
```

## 开发优先级建议

### MVP 版本（第一、二阶段）

优先交付可交易闭环：
1. 用户登录
2. 画廊浏览
3. 下单购买
4. 基础订单管理

### 社交版本（第三阶段）

增加分销能力：
1. 艺荐官体系
2. 二级分销
3. 佣金体系

### 完整版本（第四至六阶段）

完整功能矩阵：
1. 拍卖模块
2. 社区功能
3. 运营工具
