## 二、一键启动

### 方式一：使用一键脚本（推荐）

```bash
cd /Users/master/CodeBuddy/art12

# 一键启动所有服务
./scripts/start-all.sh

# 查看服务健康状态
./scripts/health-check.sh

# 一键停止所有服务
./scripts/stop-all.sh
```

### 方式二：手动启动

```bash
cd /Users/master/CodeBuddy/art12

# 1. 启动Nacos
cd nacos && sh bin/startup.sh -m standalone &
sleep 15

# 2. 启动微服务
cd ../backend
for jar in shiyiju-gateway shiyiju-user shiyiju-product shiyiju-order \
           shiyiju-auction shiyiju-promotion shiyiju-community \
           shiyiju-file shiyiju-message shiyiju-admin; do
    java -jar ${jar}/target/${jar}-1.0.0-SNAPSHOT.jar > /tmp/${jar}.log 2>&1 &
done

# 3. 启动前端
cd ../frontend && npm run dev:h5 &
cd ../admin && npm run dev &
```

  app-id: your-app-id

  api-key: your-api-key
  notify-url: http://your-domain/api/order/pay/callback
