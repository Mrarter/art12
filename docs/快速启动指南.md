## 🚀 启动方式一：IDE启动（推荐）

### IntelliJ IDEA
1. 打开项目 `/Users/master/CodeBuddy/art12`
2. 导入 `backend/pom.xml` 为Maven项目
3. 按顺序启动以下服务：

#### 启动顺序
1. **shiyiju-gateway** (网关服务 - 端口 8080)
   - 配置: `backend/shiyiju-gateway/src/main/resources/application.yml`
   - 类: `ShiyijuGatewayApplication.java`

2. **shiyiju-user** (用户服务 - 端口 8081)
   - 配置: `backend/shiyiju-user/src/main/resources/application.yml`
   - 类: `ShiyijuUserApplication.java`

3. **shiyiju-product** (商品服务 - 端口 8082)
   - 配置: `backend/shiyiju-product/src/main/resources/application.yml`
   - 类: `ShiyijuProductApplication.java`

4. **shiyiju-order** (订单服务 - 端口 8083)
   - 配置: `backend/shiyiju-order/src/main/resources/application.yml`
   - 类: `ShiyijuOrderApplication.java`

#### 启动前端
```bash
cd frontend
npm install  # 如需安装依赖
npm run dev:h5
```

## 🚀 启动方式二：命令行启动

### 后端服务
```bash
cd backend

# 打包所有服务
mvn clean package -DskipTests

# 启动网关
java -jar shiyiju-gateway/target/shiyiju-gateway.jar

# 新开终端，启动用户服务
java -jar shiyiju-user/target/shiyiju-user.jar

# 新开终端，启动商品服务
java -jar shiyiju-product/target/shiyiju-product.jar

# 新开终端，启动订单服务
java -jar shiyiju-order/target/shiyiju-order.jar
```

### 前端服务
```bash
# 新开终端
cd frontend
npm run dev:h5
```

## 🐛 常见问题

### 问题1: 端口被占用
```bash
# 查看端口占用
lsof -i :8080
lsof -i :3306

# 杀死占用进程
kill -9 <PID>
```

### 问题2: 数据库连接失败
```bash
# 检查MySQL服务
brew services restart mysql@8.0

# 测试连接
mysql -u root -proot123456
```

### 问题3: 前端无法启动
```bash
cd frontend
rm -rf node_modules
npm install
npm run dev:h5
```

### 问题4: 编译失败
```bash
cd backend
mvn clean install -DskipTests
```
