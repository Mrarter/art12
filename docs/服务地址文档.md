## 💻 服务启动命令

### 1. MySQL 数据库
```bash
# 启动 MySQL
mysql.server start

# 验证连接
mysql -u root -proot -e "SHOW DATABASES;"
```

### 2. 后台管理后端
```bash
cd /Users/master/CodeBuddy/art12/backend
java -jar shiyiju-admin/target/shiyiju-admin-1.0.0-SNAPSHOT.jar
# 启动后访问: http://localhost:8090
```

### 3. 后台管理前端
```bash
cd /Users/master/CodeBuddy/art12/admin
```

## ⚠️ 注意事项

1. **数据库连接**: 确保 MySQL 已启动并创建 `shiyiju` 数据库
2. **端口占用**: 确保所需端口 (8090, 5174, 3306) 未被其他应用占用
3. **跨域配置**: 前端通过 Vite 代理访问后端 API，无需额外配置 CORS
4. **微信小程序**: 需要在微信开发者工具中导入 `test-miniprogram` 目录
