## 🚀 快速开始

### 方式一：使用Docker（推荐）✅

#### 步骤1：安装Docker Desktop

下载地址：https://www.docker.com/products/docker-desktop

#### 步骤2：启动基础设施服务

```bash
# 1. 进入部署目录
cd deploy/docker

# 2. 启动基础设施（MySQL、Redis、Nacos）
docker-compose -f docker-compose.dev.yml up -d

# 3. 等待服务启动（约30秒）
docker ps
```

#### 步骤3：初始化数据库

```bash
# 进入项目根目录
cd /Users/master/CodeBuddy/art12

# 导入基础数据库表
mysql -h 127.0.0.1 -u root -proot123456 shiyiju < documents/db/shiyiju_user.sql
mysql -h 127.0.0.1 -u root -proot123456 shiyiju < documents/db/shiyiju_product.sql
mysql -h 127.0.0.1 -u root -proot123456 shiyiju < documents/db/shiyiju_order.sql
mysql -h 127.0.0.1 -u root -proot123456 shiyiju < documents/db/shiyiju_auction.sql
mysql -h 127.0.0.1 -u root -proot123456 shiyiju < documents/db/shiyiju_promotion.sql
mysql -h 127.0.0.1 -u root -proot123456 shiyiju < documents/db/shiyiju_community.sql
mysql -h 127.0.0.1 -u root -proot123456 shiyiju < documents/db/shiyiju_message.sql

# 导入新增功能表
mysql -h 127.0.0.1 -u root -proot123456 shiyiju < backend/shiyiju-file/sql/file_info.sql
mysql -h 127.0.0.1 -u root -proot123456 shiyiju < backend/shiyiju-product/sql/review.sql
mysql -h 127.0.0.1 -u root -proot123456 shiyiju < backend/shiyiju-order/sql/logistics.sql
```

#### 步骤4：启动后端服务

```bash
# 进入后端目录
cd backend

# 方式A：使用Maven启动（仅启动网关和用户服务作为示例）
mvn spring-boot:run -pl shiyiju-gateway -am

# 方式B：使用IDE启动
# 1. 打开 IDEA
# 2. 导入 backend/pom.xml 为Maven项目
# 3. 依次启动以下服务（按顺序）：
#    - Nacos（如果本地安装）
#    - shiyiju-gateway (8080)
#    - shiyiju-user (8081)
#    - shiyiju-product (8082)
#    - shiyiju-order (8083)
#    - shiyiju-auction (8084)
#    - shiyiju-promotion (8085)
#    - shiyiju-community (8086)
#    - shiyiju-file (8087)
#    - shiyiju-message (8088)
#    - shiyiju-admin (8090)
```

#### 步骤5：启动前端服务

```bash
# 新开终端窗口
cd frontend

# 安装依赖
npm install

# 启动H5开发服务器
npm run dev:h5

# 或者启动微信小程序开发模式
npm run dev:mp-weixin
```

#### 步骤6：访问应用

- **H5应用**：http://localhost:5173
- **管理后台**：http://localhost:5174
- **Nacos控制台**：http://localhost:8848/nacos (nacos/nacos)
- **API网关**：http://localhost:8080

### 方式二：本地安装（不使用Docker）⚠️

#### 步骤1：安装MySQL

**macOS：**
```bash
# 使用Homebrew安装
brew install mysql
brew services start mysql

# 设置root密码
mysql_secure_installation
mysql -u root -p -e "ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'root123456';"
```

**Windows：**
从 https://dev.mysql.com/downloads/mysql/ 下载安装包

#### 步骤2：安装Redis

**macOS：**
```bash
brew install redis
brew services start redis
```

**Windows：**
使用 https://github.com/microsoftarchive/redis/releases 下载

#### 步骤3：安装Nacos

```bash
# 下载Nacos
cd /tmp
wget https://github.com/alibaba/nacos/releases/download/2.2.3/nacos-server-2.2.3.tar.gz
tar -xvf nacos-server-2.2.3.tar.gz
mv nacos /opt/

# 单机模式启动
cd /opt/nacos/bin
./startup.sh -m standalone
```

#### 步骤4：配置数据库

```bash
mysql -u root -p

# 创建数据库
CREATE DATABASE IF NOT EXISTS shiyiju CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

## 🔧 常见问题

### Q1：端口被占用

**解决方案：**
```bash
# 查找占用端口的进程
lsof -i :3306
lsof -i :6379
lsof -i :8080
lsof -i :5173

# 杀死进程（替换 PID 为进程号）
kill -9 PID
```

### Q2：MySQL连接失败

**解决方案：**
1. 检查MySQL服务是否运行
   ```bash
   mysqladmin ping -u root -proot123456
   ```

2. 检查用户权限
   ```sql
   mysql -u root -p
   CREATE USER 'root'@'%' IDENTIFIED BY 'root123456';
   GRANT ALL PRIVILEGES ON shiyiju.* TO 'root'@'%';
   FLUSH PRIVILEGES;
   ```

### Q3：Redis连接失败

**解决方案：**
```bash
# 检查Redis服务
redis-cli ping

# 如果需要密码
redis-cli -a your_password ping
```

### Q4：Nacos启动失败

**解决方案：**
```bash
# 查看日志
cat /opt/nacos/logs/start.out

# 检查端口
lsof -i :8848
```

### Q5：前端启动失败

**解决方案：**
```bash
# 清理node_modules
rm -rf node_modules package-lock.json

# 重新安装
npm install

# 使用淘宝镜像（国内加速）
npm install --registry=https://registry.npmmirror.com
```

### Q6：编译后端失败

**解决方案：**
```bash
# 清理Maven缓存
cd backend
mvn clean

# 重新编译
mvn compile

# 或者跳过测试
mvn clean install -DskipTests
```

## 🎯 开发建议

### 1. 使用IDE启动微服务

**IntelliJ IDEA：**
- 导入 `backend/pom.xml` 为Maven项目
- 使用Spring Boot运行配置
- 设置环境变量或使用 `.env` 文件

### 2. 前后端联调

1. 启动后端所有服务
2. 启动前端开发服务器
3. 使用浏览器访问 http://localhost:5173
4. 打开浏览器开发者工具查看网络请求

### 3. 微信小程序调试

1. 启动后端服务
2. 执行 `npm run dev:mp-weixin`
3. 使用微信开发者工具导入 `frontend/dist/dev/mp-weixin`
4. 配置不校验合法域名（开发阶段）

### 4. 数据库管理

推荐使用以下数据库客户端：
- **DBeaver**（免费，支持多数据库）
- **Navicat**（功能强大）
- **DataGrip**（JetBrains出品）
