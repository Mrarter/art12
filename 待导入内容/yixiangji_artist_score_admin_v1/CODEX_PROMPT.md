# Codex 开发提示词：Vue3 后台管理系统评分审核与调控模块

你正在开发「艺享集 / 拾艺局」艺术品交易平台的后台管理系统。

## 技术栈

- Vue3
- Vite
- Element Plus
- Axios
- Vue Router
- Pinia 可选

## 本次任务

开发艺术家评分审核与作品价格调控模块。

## 页面

1. 艺术家评分列表
2. 艺术家资质审核页
3. 艺术家评分详情页
4. 作品价格调控页
5. 作品价格日志页

## 关键要求

- 所有审核动作都要调用后端接口
- 审核通过后调用重新计算评分接口
- 人工调分必须填写原因
- 作品价格人工调整必须写入价格日志
- 不允许前端直接计算最终分数
- 前端只做展示和操作提交

## 菜单建议

```text
艺术家管理
├── 艺术家评分
├── 资质审核
└── 评分详情

交易调控
├── 价格调控
└── 价格日志
```
