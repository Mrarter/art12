# 后台操作流程

## 一、艺术家提交资质

艺术家在小程序端或管理端提交：

- 毕业院校
- 学历
- 职称
- 协会身份
- 展览经历
- 获奖经历
- 抖音/小红书/视频号账号
- 粉丝截图或链接
- 证明材料图片

进入后台状态：

```text
PENDING 待审核
```

---

## 二、后台审核资质

运营人员进入：

```text
艺术家管理 / 资质审核
```

审核动作：

- 通过
- 驳回
- 添加备注

通过后：

```text
verified = 1
audit_status = PASS
自动触发评分重算
```

---

## 三、查看艺术家评分

进入：

```text
艺术家管理 / 艺术家评分
```

可查看：

- 总分
- 等级
- 学术资质分
- 互联网资质分
- 更新时间

---

## 四、人工调分

只有管理员可以操作。

必须填写：

- 调整分值
- 调整原因

建议后端额外记录：

- 操作人
- 操作时间
- 原分数
- 新分数

---

## 五、作品价格调控

进入：

```text
交易调控 / 价格调控
```

可以查看：

- 当前价格
- 收藏数
- 成交数
- 状态

人工调价必须填写原因，并写入价格日志。

---

## 六、价格日志

进入：

```text
交易调控 / 价格日志
```

查看所有价格变化：

- 每日涨价
- 成交触发
- 收藏触发
- 评分触发
- 人工调价
