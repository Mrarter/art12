# 小程序页面流程

## 艺术家主页

进入路径：

```text
/pages/artist-profile/artist-profile?artistId=1
```

页面展示：

- 艺术家头像
- 艺术家名称
- 艺术家等级
- 总评分
- 资质标签
- 成长指数
- 查看评分详情按钮

---

## 艺术家评分详情页

进入路径：

```text
/pages/artist-score/artist-score?artistId=1
```

页面展示：

- 总等级
- 总分
- 七个评分维度
- 评分说明

---

## 作品详情页

进入路径：

```text
/pages/artwork-detail/artwork-detail?artworkId=1
```

页面展示：

- 作品图片
- 作品名称
- 艺术家名称
- 当前价格
- 今日涨幅
- 收藏热度
- 下一次涨价条件
- 收藏按钮
- 立即购买按钮
