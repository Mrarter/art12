## 产品概述

为艺术家管理（artist.vue）和艺荐官管理（promoter.vue）两个后台管理页面添加搜索、筛选、排序、导出及批量操作功能。

## 核心功能

- **搜索栏**：支持按昵称/真实姓名、手机号、ID/UID、认证等级/艺荐官等级进行搜索
- **高级筛选**：支持按时间范围（申请时间/成为时间）筛选
- **列表排序**：支持按时间、累计佣金等字段排序（点击表头切换升序/降序）
- **数据导出**：支持将当前列表数据导出为 CSV 文件
- **艺荐官批量操作**：添加批量通过、批量拒绝、批量删除功能（与艺术家管理页面保持一致）

## 技术栈

- 前端：Vue 3 + Element Plus + Vite
- 后端：Spring Boot + Spring JDBC Template
- 请求层：自定义 request 封装（admin/src/api/request.js）

## 实现方案

### 后端改造

#### 1. UserManagementController.java 接口参数扩展

**艺术家列表接口** `/admin/user/artist/list`：

- 现有参数：`page`, `size`, `status`
- 新增参数：
- `keyword`（昵称/姓名模糊搜索）
- `phone`（手机号精确匹配）
- `userId`（用户ID/UID）
- `badge`（认证等级筛选）
- `startDate`（开始时间）
- `endDate`（结束时间）
- `sortField`（排序字段）
- `sortOrder`（排序方向：asc/desc）

**艺荐官列表接口** `/admin/user/promoter/list`：

- 现有参数：`page`, `size`, `userId`, `level`, `status`
- 新增参数：
- `keyword`（昵称/姓名模糊搜索）
- `phone`（手机号精确匹配）
- `startDate`（开始时间）
- `endDate`（结束时间）
- `sortField`（排序字段）
- `sortOrder`（排序方向）

**新增艺荐官批量接口**：

- `POST /admin/user/promoter/batchApprove` - 批量通过
- `POST /admin/user/promoter/batchReject` - 批量拒绝
- `POST /admin/user/promoter/batchDelete` - 批量删除

#### 2. UserAdminPersistenceService.java 方法改造

**listArtists() 方法**：

- 新增参数：`keyword`, `phone`, `userId`, `badge`, `startDate`, `endDate`, `sortField`, `sortOrder`
- 动态构建 WHERE 子句（模糊搜索 nickname/realName，精确匹配 phone/userId，badge 匹配）
- 动态构建 ORDER BY 子句（白名单校验排序字段，防止 SQL 注入）

**listPromoters() 方法**：

- 新增参数：`keyword`, `phone`, `startDate`, `endDate`, `sortField`, `sortOrder`
- 动态构建 WHERE 子句
- 动态构建 ORDER BY 子句

**新增批量方法**：

- `batchApprovePromoters(List<Long> ids)` - 批量通过艺荐官认证
- `batchRejectPromoters(List<Long> ids, String reason)` - 批量拒绝
- `batchDeletePromoters(List<Long> ids)` - 批量删除

### 前端改造

#### 3. artist.vue 改造

**搜索栏**（放在状态 Tab 下方、表格上方）：

- 使用 Element Plus `el-form` 内联表单
- 搜索字段：昵称/姓名输入框、手机号输入框、ID/UID输入框、认证等级下拉选择、搜索/重置按钮
- 时间范围：日期范围选择器（申请时间）

**排序功能**：

- 在 `el-table-column` 上添加 `sortable="custom"` 属性
- 监听 `@sort-change` 事件，将排序参数传给后端

**CSV 导出**：

- 新增"导出"按钮（页面右上角）
- 点击后调用后端接口获取全量数据（或当前筛选条件下数据），前端拼接 CSV 并触发下载

#### 4. promoter.vue 改造

**搜索栏**：与 artist.vue 保持一致风格

- 搜索字段：昵称/姓名、手机号、ID/UID、等级下拉选择、搜索/重置按钮
- 时间范围：日期范围选择器（成为时间）

**批量操作栏**：

- 添加 `el-table-column type="selection"` 多选列
- 添加批量操作栏（参考 artist.vue 实现）：显示已选数量、批量通过/拒绝/删除按钮
- 添加批量拒绝原因弹窗

**排序功能**：同 artist.vue

**CSV 导出**：同 artist.vue

## 设计风格

保持与现有后台管理系统一致的 Element Plus 风格，搜索区域采用卡片式布局，与表格区域视觉层次分明。

## 页面布局设计

### 艺术家管理页（artist.vue）

**搜索区域块**（状态 Tab 下方）：

- 使用 `el-card` 包裹搜索表单，卡片标题为"搜索筛选"
- 第一行：昵称/姓名输入框（width: 180px）、手机号输入框（width: 150px）、ID/UID输入框（width: 150px）
- 第二行：认证等级下拉（全部/普通认证/认证艺术家/人气艺术家/大师级）、时间范围选择器、搜索按钮、重置按钮
- 搜索按钮为主色调，重置按钮为默认色调

**表格区域增强**：

- 昵称列、认证等级列、申请时间列添加 `sortable="custom"` 属性
- 表格右上角添加"导出CSV"按钮（小号，link类型）

**导出弹窗**：

- 点击导出后弹出确认弹窗，显示"将导出当前筛选条件下的 N 条数据"
- 确认后前端生成 CSV 并触发浏览器下载

### 艺荐官管理页（promoter.vue）

**搜索区域块**（状态 Tab 下方）：

- 与艺术家页面保持一致布局
- 第一行：昵称/姓名输入框、手机号输入框、ID/UID输入框
- 第二行：等级下拉（全部/普通/高级/金牌）、时间范围选择器、搜索按钮、重置按钮

**批量操作栏**（参考 artist.vue 实现）：

- 选中行后显示批量操作栏（浅蓝色背景）
- 显示"已选择 N 个艺荐官"
- 按钮：批量通过（绿色）、批量拒绝（红色）、批量删除（红色）
- 批量拒绝需要弹窗填写拒绝原因

**表格区域增强**：与艺术家页面一致

## 交互说明

- 搜索表单支持回车触发搜索
- 时间范围选择器支持快捷选项（今天/本周/本月）
- 排序：点击表头切换 不排序/升序/降序 三种状态
- CSV 导出：使用 `\uFEFF` BOM 头确保中文编码正确
